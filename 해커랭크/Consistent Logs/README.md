### 입출력 예 설명

**입출력 예 #1**

```
userEvent = [9, 9, 9]
전체 배열에서 9는 3번 등장

부분배열 [9, 9, 9]에서도 9는 3번 등장

최대 빈도(3) = 최소 빈도(3) ✓

답: 3

```

**입출력 예 #2**

```
userEvent = [9, 8, 5, 9, 2]
전체 배열 빈도:

9: 2번
8: 1번
5: 1번
2: 1번

최소 빈도 = 1
부분배열 [8, 5, 9, 2]:

각 ID가 1번씩만 등장
최대 빈도(1) = 최소 빈도(1) ✓

답: 4
```

**입출력 예 #3**

```
userEvent = [1, 2, 1, 3, 4, 2, 4, 3, 3, 4]
전체 배열 최소 빈도 = 2 (ID 1, 2가 2번씩)
부분배열 [1, 2, 1, 3, 4, 2, 4, 3]:

모든 ID가 2번씩 등장
최대 빈도(2) = 최소 빈도(2) ✓
길이: 8

답: 8
```
