# 부분 배낭 문제

### 문제 유형

Greedy | 탐욕법

### 문제 설명

<p>무게와 가치가 있는 짐items와 배낭 weightLimit이 주어질때 부분 배낭 문제를 푸는 solution()함수를 작성해주세요.
</p>

<h5>제약 조건</h5>

<ul>
<li>weightLimit은 1이상 10,000이하의 자연수입니다.</li>
<li>items의 길이는 1이상 1,000이하입니다.</li>
</li>
