# 두 개 뽑아서 더하기

### 구분

해시 Set 문제 (feat.Claude AI)

### 문제 설명

<p>정정수 배열 numbers가 주어집니다.
서로 다른 인덱스의 두 수를 뽑아 더한 결과를 오름차순 배열로 반환하세요.</p>

<h5>제한 조건</h5>

<ul>
    <li>2 ≤ numbers.length ≤ 100</li>
    <li>0 ≤ numbers[i] ≤ 100</li>
</ul>

<h5>입출력 예</h5>

<table class="table">
    <thead>
        <tr>
            <th>numbers</th>
            <th>results</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>[2,1,3,4,1]</td>
            <td>[2,3,4,5,6,7]</td>
        </tr>
         <tr>
            <td>[5,0,2,7]</td>
            <td>[2,5,7,9,12]</td>
        </tr>
    </tbody>
</table>

### 입출력 예 설명

**입출력 예 #1**

```
[2,1,3,4,1]

가능한 합:
2+1=3, 2+3=5, 2+4=6, 2+1=3
1+3=4, 1+4=5, 1+1=2
3+4=7, 3+1=4
4+1=5

중복 제거 + 정렬: [2,3,4,5,6,7]
```

**입출력 예 #2**

```
nums = [3, 2, 4], target = 6

3 + 2 = 5 ❌
3 + 4 = 7 ❌
2 + 4 = 6 ✓

인덱스: [1, 2]
답: [1, 2]
```

**입출력 예 #3**

```
nums = [3, 3], target = 6

3 + 3 = 6 ✓
인덱스: [0, 1]

답: [0, 1]
```

**입출력 예 #3**

```
heights = [10, 20, 10], k = 1

k=1이므로 1칸씩만!

경로: 0 → 1 → 2
에너지: |20-10| + |10-20| = 10 + 10 = 20 ✓

답: 20
```
