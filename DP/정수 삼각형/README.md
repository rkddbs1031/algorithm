# 정수 삼각형

### 구분

DP (2차원) (feat.Claude AI)

### 문제 설명

<p>삼각형 모양으로 숫자가 배치되어 있습니다.</p>
<p>맨 위에서 시작하여 아래로 내려가면서 숫자를 선택합니다.</p>
<p><strong>규칙:</strong> 대각선 아래로만 이동 가능 (바로 아래 또는 바로 아래 오른쪽)</p>
<p>선택한 숫자들의 <strong>합의 최댓값</strong>을 return 하세요.</p>

<h5>제한 조건</h5>

<ul>
    <li>1 ≤ 삼각형 높이 ≤ 500</li>
    <li>0 ≤ 숫자 ≤ 9,999</li>
</ul>

<h5>입출력 예</h5>
<table class="table">
    <thead>
        <tr>
            <th>triangle</th>
            <th>result</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>[[7], [3, 8], [8, 1, 0], [2, 7, 4, 4], [4, 5, 2, 6, 5]]</td>
            <td>30</td>
        </tr>
    </tbody>
</table>

### 입출력 예 설명

**입출력 예 #1**

```
        7
       3 8
      8 1 0
     2 7 4 4
    4 5 2 6 5

최적 경로: 7 → 3 → 8 → 7 → 5 = 30
```

<h5>이동 규칙</h5>
 [i][j]에서
→ [i+1][j] (바로 아래)
→ [i+1][j+1] (아래 오른쪽)
