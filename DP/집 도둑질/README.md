# 집 도둑질

### 구분

DP 연습 문제 (feat.Claude AI)

### 문제 설명

<p>당신은 전문 도둑입니다. 일렬로 늘어선 집들을 털 계획인데, 각 집에는 일정 금액의 돈이 있습니다.</p>

<p>
문제는 <strong>인접한 두 집을 연속으로 털면 경보가 울린다</strong>는 것입니다. 
즉, i번 집을 털었다면 (i-1)번이나 (i+1)번 집은 털 수 없습니다.
</p>
<p>
각 집에 있는 돈이 주어질 때, 경보를 울리지 않고 훔칠 수 있는 <strong>최대 금액</strong>을 구하세요.
</p>

<h5>제한 조건</h5>

<ul>
    <li>1 ≤ <code>nums.length</code> ≤ 100</li>
    <li>0 ≤ <code>nums[i]</code> ≤ 400</li>
    <li><code>nums[i]</code>는 i번째 집에 있는 돈입니다.</li>
</ul>

<h5>입출력 예</h5>
<table class="table">
    <thead>
        <tr>
            <th>nums</th>
            <th>result</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>[1, 2, 3, 1]</td>
            <td>4</td>
        </tr>
        <tr>
            <td>[2, 7, 9, 3, 1]</td>
            <td>12</td>
        </tr>
        <tr>
            <td>[5, 3, 4, 11, 2]</td>
            <td>16</td>
        </tr>
        <tr>
            <td>[1, 3, 1, 3, 100]</td>
            <td>103</td>
        </tr>
    </tbody>
</table>

### 입출력 예 설명

**입출력 예 #1**

```
집:  [1, 2, 3, 1]
     0  1  2  3

경로 1: 0번(1) + 2번(3) = 4 ✓
경로 2: 1번(2) + 3번(1) = 3
경로 3: 0번(1) + 3번(1) = 2

답: 4
```

**입출력 예 #2**

```
집:  [2, 7, 9, 3, 1]
     0  1  2  3  4

경로 1: 0번(2) + 2번(9) + 4번(1) = 12 ✓
경로 2: 1번(7) + 3번(3) = 10
경로 3: 0번(2) + 2번(9) = 11

답: 12
```

**입출력 예 #3**

```
집:  [5, 3, 4, 11, 2]
     0  1  2  3   4

경로: 0번(5) + 3번(11) = 16 ✓

답: 16
```

**입출력 예 #4**

```
집:  [1, 3, 1, 3, 100]
     0  1  2  3  4

경로: 1번(3) + 4번(100) = 103 ✓

답: 103
```
