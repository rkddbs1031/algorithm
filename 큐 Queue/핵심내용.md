## Queue (큐)

- 자료구조 : FIFO(First In First Out)
- 먼저 들어간 것이 먼저 나옴
- 예시: 줄 서기, 프린터 대기열, BFS

## 주요 연산:

- push(item): 맨 뒤에 추가 O(1)
- pop(): 맨 앞 제거하고 반환 O(1)
- isEmpty(): 비어있는 지 확인 O(1)
- size(): 크기 반환 - O(1)

## queue 직접 구현

- 배열 자체는 안 건드리고, front/rear `포인터`만 움직이기!

```
class Queue {
     constructor() {
        this.items = [];
        this.front = 0;
        this.rear = 0;
    }

    push(item) { // enqueue
        this.items.push(item);
        this.rear++;
    }

    pop(){ // dequeue
        if (this.isEmpty()) return null;
        return this.items[this.front++];
    }

    isEmpty(){
        return this.front === this.rear;
    }

    size() {
        return this.rear - this.front;
    }
}

```

## 실전 활용:

- **BFS (너비 우선 탐색)** - 그래프/트리 탐색
- **최근 N개 추적** - 슬라이딩 윈도우
- **작업 스케줄링** - 프로세스 관리
- **캐시 구현** - LRU Cache
- **레벨별 처리** - 트리의 레벨 순회
