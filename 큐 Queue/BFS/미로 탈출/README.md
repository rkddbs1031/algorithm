# 미로 탈출

### 구분

BFS (Queue) / 최단 거리 (feat.Claude AI)

### 문제 설명

<p>n x m 크기의 미로가 있습니다.</p>
<p>1은 길, 0은 벽입니다.</p>
<p>(0, 0)에서 시작해서 (n-1, m-1)까지 가는 <strong>최단 거리</strong>를 return 하세요.</p>
<p>상하좌우로만 이동 가능합니다.</p>

<h5>제한 조건</h5>

<ul>
    <li>2 ≤ n, m ≤ 100</li>
    <li>시작점과 도착점은 항상 1 (갈 수 있음)</li>
    <li>도착할 수 없으면 -1 반환</li>
</ul>

<h5>입출력 예</h5>

<table class="table">
    <thead>
        <tr>
            <th>maze</th>
            <th>result</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>[[1,1,0], [1,1,0], [0,1,1]]</td>
            <td>5</td>
        </tr>
        <tr>
            <td>[[1,0,1], [1,1,1], [0,0,1]]</td>
            <td>5</td>
        </tr>
        <tr>
            <td>[[1,0], [0,1]]</td>
            <td>-1</td>
        </tr>
    </tbody>
</table>

**입출력 예 #1**

```
1 1 0
1 1 0
0 1 1

경로: (0,0) → (0,1) → (1,1) → (2,1) → (2,2)
거리: 5칸
```

**입출력 예 #2**

```
1 0 1
1 1 1
0 0 1

경로: (0,0) → (1,0) → (1,1) → (1,2) → (2,2)
거리: 5칸
```

**입출력 예 #3**

```
1 0
0 1

막혀있음 → -1
```

<h5>힌트</h5>

<ul>
    <li>큐에 [x, y, 거리] 저장</li>
    <li>visited로 방문 체크</li>
    <li>상하좌우 4방향 탐색</li>
    <li>도착점 도달하면 거리 반환</li>
</ul>
