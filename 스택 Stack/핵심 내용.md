## Stack (스택)

- 자료구조: LIFO (Last In First Out)
- 마지막에 들어간 것이 먼저 나옴
- 예시: 접시 쌓기, 뒤로가기 버튼, 함수 호출 스택

## 주요 연산:

- push(item): 맨 위에 추가 - O(1)
- pop(): 맨 위 제거하고 반환 - O(1)
- peek(): 맨 위 확인 (제거 안 함) - O(1)
- isEmpty(): 비어있는지 확인 - O(1)
- size(): 크기 반환 - O(1)

## stack 직접 구현

- queue와 달라 front, rear이 필요없음!

```
class Stack{
    // items = []; // 최신 문법

    constructor() {
        this.items = [];
    }

    push(item) {
        this.items.push(item);
    }

    pop(){
        if(this.isEmpty()) return null;
        return this.items.pop();
    }

    isEmpty() {
        return this.items.length === 0;
    }

    peek(){
        if (this.isEmpty()) return null;
        return this.items[this.items.length - 1];
    }

    size() {
        return this.items.length;
    }
}
```
